Inheritance is when a class has a child that that implements all of its public
functions. Polymorphic is when you can  apply the parent class and its parts onto the
derived classes and objects as well. Encapsulation is having all the related data and

With Integration Manager, the one integration manager fixes all conflicts before push to
the blessed repository. With Dictator and Lieutenants the liutenants fix the conflicts from
the developers under them before pushing to dictator, then the dictator fixes conflicts     
from the lieutenants before pushing to the blessed repository.

10.
In centralized each developer would have to manage conflicts with up to 100 other developers.
With Dictator and lieutenant, each lieutenany only need to deal with conflicts from a subset
of the developers, then the dictator only needs to deal eith conflicts from the lieutenants,
reducing conflict resolution workload on each individual.
